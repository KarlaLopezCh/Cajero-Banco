# Proyecto: Interfaz de Cajero Automático - Banco Ficticio

Este proyecto es una simulación de la interfaz gráfica de un cajero automático (ATM) para un banco ficticio. Fue desarrollado como parte del curso para practicar HTML, CSS y JavaScript, enfocándose en diseño de interfaz y experiencia de usuario (UI/UX).

## Características

- Pantalla de bienvenida  
- Inicio de sesión con clave de cajero (simulado)  
- Menú principal con opciones:  
  - Consultar saldo  
  - Transferencias  
  - Ingresar dinero  
  - Retirar dinero  
- Pantallas individuales para cada acción  
- Diseño minimalista y adaptable  

## Tecnologías utilizadas

- **HTML5** – Estructura del proyecto  
- **CSS3** – Estilos y diseño visual  
- **JavaScript** – Lógica interactiva de la simulación  

## Cómo usar el proyecto

1. Abrir el archivo `index.html` en el navegador.  
2. Desde la pantalla de inicio, seguir las instrucciones para navegar entre las distintas funciones del cajero.  
3. Las operaciones son simuladas; no hay transacciones reales.

# Project: ATM Interface - Fictional Bank

This project is a simulation of the user interface of an Automated Teller Machine (ATM) for a fictional bank. It was developed as part of a course to practice HTML, CSS, and JavaScript, focusing on interface design and user experience (UI/UX).

## Features

- Welcome screen  
- Login screen with ATM password (simulated)  
- Main menu with options:  
  - Check balance  
  - Transfer funds  
  - Deposit money  
  - Withdraw money  
- Individual screens for each operation  
- Minimal and clean interface design  

## Technologies Used

- **HTML5** – Structure of the project  
- **CSS3** – Visual design and styles  
- **JavaScript** – Logic for interactivity and screen transitions  

## How to Use the Project

1. Open the `index.html` file in your web browser.  
2. From the welcome screen, follow the interface instructions to access the different ATM functions.  
3. All operations are simulated; no real transactions are performed.
